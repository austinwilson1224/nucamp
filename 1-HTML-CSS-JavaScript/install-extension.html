Installation instructions

  1. In VS Code, click the extensions icon in toolbar.
  2. In the search bar that appears, type live server.
  3. Select the live server extension. It should be on top
  4. In the page for live server that appears, click the green install button

Usage instructions

  • VS code must be open to a folder to use live server
  • in VS code's explorer, right click (windown) or ctrl click. From context menu select open w/ live server
